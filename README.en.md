# gulimall

## 项目名
电商商城

## 项目介绍

1.  该项目实现了手机号验证注册及登录功能采（用阿里云发送短信）.
2.  商品搜索功能（ElasticSearch检索）.
3.  商品上架功能.
4.  购物车功能（临时购物车、用户购物车）.

## 使用技术
1.基于微服务开发、Spring Boot、Spring Cloud、等技术...
2.前后端分离.
3.通过Nginx反向代理网关，通过网关路由到每个微服务模块业务上.
